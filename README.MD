# 物业管理系统

## 项目要求

1.某写字楼物业管理公司管理着一栋3层的办公楼，每层分A、B两座，每座有5套房间。

2.编写程序，输入楼层数、每层座数以及每座房间数，自动生成所有的房间及其编号。

3.用户可以增加、删除、修改房间及其编号，设置每一个房间的住户及其电话号码。

4.可以根据住户姓名查找其房间及编号，可以列出所有或指定楼层、座次、房间的住户信息。

## 功能实现

1.对三层办公楼的AB两座进行管理

2.房间编号自动生成

    ①输入楼层数
    
    ②输入每层的座数

    ③输入每座的房间数

    ④自动生成所有房间的编号

3.房间信息编辑系统

    ①增加，修改，删除房间机器编号

    ②设置每个房间的住户及信息

4.查询系统查询系统

    ①查找住户姓名

    ②查找用户房间及其编号

    ③查询所有楼层的座次，房间的住户信息

    ④查询指定楼层的座次，房间的住户信息
